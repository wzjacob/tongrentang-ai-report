同仁堂集团 · 算力摸排与 AI 前瞻规划汇报

本页为本地快捷入口。完整汇报为 Next.js 项目，需先在本机启动开发服务后再打开。

  1. 在资源管理器中进入本 HTML 所在文件夹（项目根目录 tongrentang-ai-report）。
  2. 打开终端，执行：npm install（首次）然后 npm run dev。
  3. 看到服务就绪后，点击下方按钮在浏览器中打开汇报。
打开汇报（127.0.0.1:3333） 备用地址（localhost:3333）
若点击后无法打开：请确认终端里 npm run dev 已正常运行，且端口为 3333（与 package.json 中配置一致）。
文件位置：与 package.json 同目录，可随意复制到桌面使用。
